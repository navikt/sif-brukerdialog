# Ettersending av dokumenter innenfor sykdom i familien

Kildekod til frontend-appen for ettersending av dokumenter innenfor sykdom i familien.

## Komme i gang

For å kjøre opp app-en i dev:

1.  `yarn`.
2.  `yarn dev`.

For å kjøre enhetstester:

1.  `yarn && yarn test`.
